# Steam OpenID Login App

Este é um app básico em Flask que permite o login de usuários via Steam e exibe o SteamID64.

## Como usar

1. Instale o Python 3 (https://www.python.org/)
2. No terminal/cmd, instale as dependências:

```
pip install flask requests
```

3. Rode o aplicativo:

```
python app.py
```

4. Acesse http://localhost:5000 no navegador.

## Estrutura

- `app.py`: código principal
- `README.txt`: instruções
